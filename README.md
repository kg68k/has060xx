# HAS060X.X

HAS060.X version 3.09+91の改造版です。  
無保証につき各自の責任で使用して下さい。


## 新機能

* HUPAIRに対応。
* 実行ファイル名の先頭を'g2as'(大文字小文字区別なし)にするとg2asモードで動作する。
  * 環境変数HASではなくG2ASを参照する。
  * `-1 -c4`オプションが自動指定される。
* 環境変数の先頭に'*'がなくても、常に'/'をファイル名の指定として扱う。
* コマンドラインスイッチ`-y[1]`を指定したときにプレデファインシンボル`__HAS060X__`
  (HAS060X.Xのバージョン)が定義される。
  * ただし、`-8`が同時に指定されたときは定義されない。
* 1行の最大バイト数を8KBに増加。
* `MOVEM`命令のレジスタリストを即値形式でも記述できる。


## 削除された機能

* 別名命令`CLR An`。

## Build

PCやネット上での取り扱いを用意にするために、src/内のファイルはUTF-8で記述されています。
X68000上でビルドする際には、UTF-8からShift_JISへの変換が必要です。

### src2buildを使用する場合

必要ツール: [src2build](https://github.com/kg68k/src2build)

srcディレクトリのある場所で以下のコマンドを実行します。
```
src2build src
make -C build
```

### その他の方法

src/内のファイルを適当なツールで適宜Shift_JISに変換して別のディレクトリに保存し、
ディレクトリ内で`make`を実行してください。  
UTF-8のままでは正しくビルドできません。
